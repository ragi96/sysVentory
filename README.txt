Highlights
----------

* Das GUI ist intuitiv bedienbar und reagiert schnell auf Benutzereingaben. Zudem wurde auf korrektes Tabbing durch die Form Elemente geachtet.
* In der Delta-Ansicht sind Änderungen schön mit passenden Farben hinterlegt. Dazu konnten Erfahrungen mit Richtext-Boxen gemacht werden.
* Es wurde ein einheitliches und stabiles Fehlerhandling implementiert.
* Wir konnten Erfahrungen mit der Diff-Match-Patch Library von Google machen (https://github.com/google/diff-match-patch).
* Mit der Verwendung von LINQ konnten wir einige Operationen auf Listen vereinfachen.
* Durch die Verwendung von GitHub konnten wir unkompliziert zusammenarbeiten.

Verbesserungsvorschläge
-----------------------

* Die Persistenz mit Files durch eine Datenbank ersetzen, damit die Datenzugriffe einfacher und standartisiert umgesetzt werden können.
** Wegen der Umsetzung mit JSONs würde sich z.B. eine MongoDB anbieten.
* Die View in mehrere Formulare aufteilen, da das aktuelle GUI etwas überladen ist.
* GUI Design überarbeiten, damit die Applikation optisch moderner aussieht.
